2. Installation

	Unzip the 1001236131_Sai_Kumar_Manakan.zip to any folder.

##################################################

3. Run

	Prerequisite : Make tool and Java 1.6 needs to be installed in your system

	Server:
		Navigate to <Project_Folder>/src/ and type the following command :-

			startHTTPWebServer.bat <port_number>
			or
			startHTTPWebServer.sh <port_number>

	Client:

		Navigate to <Project_Folder>/src/ and type the following command :-

			startHTTPWebClient.bat <host_name> <host_port> <resource>
			or
			startHTTPWebClient.sh <host_name> <host_port> <resource>

##################################################

4. Files

	The Implementation of the Server can be found in the Following Java Files

		<Project_Folder>/src/HTTPWebServer.java
		<Project_Folder>/src/HTTPRequest.java

	The Implementation of the Client can be found in the Following Java File

		<Project_Folder>/src/HTTPWebClient.java
